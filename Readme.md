# 🚀 Cadence — AI Powered Protection Intelligence App

**Cadence** is an AI-powered mobile application built with **React Native (Expo)** and a **Node.js backend**.

The platform helps insurance advisors and customers make smarter decisions using:

- 🤖 AI-powered recommendations
- 📊 Intelligent scoring engines
- 📱 Cross-platform React Native experience
- ⚡ Real-time backend APIs

Built for **Hackstreet — PolicyBazaar Hackathon**
